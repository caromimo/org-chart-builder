# Org Chart Builder 

This project is a library for building org charts. It leverages the [d3-org-chart](https://github.com/bumbeishvili/org-chart) library and the [Rsbuild](https://rsbuild.rs) build tool.

## Rsbuild project

### Setup

Install the dependencies:

```bash
npm install
```

### Get started

Start the dev server, and the app will be available at [http://localhost:3000](http://localhost:3000).

```bash
npm run dev
```

Build the app for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
